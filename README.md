### 1. 사용 기술

##### front-end
- 웹 표준 (HTML5, CSS3, JavaScript)
- jQuery
- BootStrap4

##### back-end
- Java, Spring Framework, JSP
- Tomcat8 서버 환경
- MySQL
- MyBatis
- node.js

##### tools
- AWS(EC2, RDS)
- STS(Eclipse)
- PuTTY
- MySQL Workbench
- eXERD

##### API
- Tmap API
- 따릉이 API
- 네아로 API
